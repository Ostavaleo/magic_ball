# MAGIC BALL
Программа "Волшебный шар" это шуточная консальная игра предсказывающая будущее.
Полное описание игры можно прочитать на Викепедии - https://ru.wikipedia.org/wiki/Magic_8_ball


## Используемый язык программирования

Программа "Волшебный шар" была написана на языке Ruby:

* [Ruby 3.0.2-1](https://rubyinstaller.org/downloads/)

   ~~~ sh
   git clone https://github.com/Ostavaleo/magic_ball.git
   ~~~

3. Откройте терминал/командную строку:


4. Перейдите в нужную директорию:
